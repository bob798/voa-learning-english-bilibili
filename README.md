# VOA Learning English｜Bilibili 学习笔记

这个仓库用于配套 Bilibili 系列视频：用 **VOA Learning English** 做可持续的英语输入、单词积累和口语模仿。

> Current series: Let's Learn English with Anna

## 为什么做这个系列

- 用真实、权威、可重复获取的 VOA Learning English 内容做英语输入。
- 每期只吃透一个短视频：听懂 → 拆句 → 学词 → 跟读 → 输出。
- 单词不做孤立背诵，而是放回原句、场景和表达功能里学习。

## 学习方法

每期建议按这个闭环学习：

1. **Watch**：先完整看 1 遍，不查词。
2. **Listen**：只听音频，抓关键词。
3. **Read**：看字幕/转写，划出核心表达。
4. **Shadow**：跟读 3 遍，模仿节奏和重音。
5. **Output**：用本期句型说/写自己的 3 句话。
